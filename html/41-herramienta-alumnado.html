Situación de Aprendizaje: ¿Dónde estamos? Aprendemos a orientarnos en nuestro entorno

4.1. Herramienta alumnado

Padlet

Para comenzar el proyecto, vais a utilizar un tablero creado con Padlet donde encontraréis todos los recursos necesarios.

En este espacio podréis acceder a distintos materiales que os ayudarán a investigar y aprender sobre la orientación y el conocimiento del entorno. Por ejemplo, encontraréis:

  • Vídeos sobre los puntos cardinales
  • Tutoriales de Google Maps
  • Recursos sobre mapas y planos
  • Páginas web para investigar monumentos de Málaga
  • Ejemplos de murales o presentaciones visuales

Podréis utilizar este tablero como guía durante todo el proyecto para consultar información, aprender cosas nuevas y ayudaros a realizar vuestro trabajo.

Aquí tenéis el enlace para acceder al Padlet:
https://padlet.com/belenmartin9/recursos-orientaci-n-y-conocimiento-del-entorno-f76rkqif664987rr
